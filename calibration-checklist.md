# 虚拟-现实一致性校准清单（独立版）

目标：让现实机器人与虚拟模型动作与姿态一致。

## A. 装配前确认
- [ ] 使用同一版模型（`cad/robot-model.scad`）导出的 STL
- [ ] 打印比例 100%，未私自缩放
- [ ] 打印件孔位清理干净，热熔铜柱已安装

## B. 舵机中位对齐（最关键）
- [ ] 每个舵机先上电到“中位”（90°）
- [ ] 再把舵机固定到结构里
- [ ] 舵盘朝向与虚拟模型 0° 对齐

## C. 线序与供电
- [ ] GPIO 线序与文档一致
- [ ] 舵机电源与 Pico 共地
- [ ] 舵机供电来自 6V 电源（不走 Pico）

## D. 初次通电测试
- [ ] 单个舵机逐一测试方向
- [ ] 若方向相反，调整 `invert=True/False`
- [ ] 若幅度不对，调整 `servo_min/servo_max`

## E. 动作一致性检查
- [ ] 左右肩抬起高度一致
- [ ] 左右肘弯折角度一致
- [ ] 夹爪开合角度一致
- [ ] `hi` 挥手动作看起来流畅

## F. 记录与锁定
- [ ] 把最终 `servo_min/servo_max` 与 `invert` 值记录下来
- [ ] 不再更改结构尺寸（避免失配）
