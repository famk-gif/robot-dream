# 机器人落地现实 第 1 阶段输出

## A. 舵机孔位模板（可直接用于建模）

### A1. MG996R 安装模板（推荐用长孔，兼容不同批次）
1. 底板尺寸：60 x 32 x 4
2. 长孔数量：4
3. 长孔尺寸：10 x 3.2
4. 长孔中心距（横向）：48.5
5. 长孔中心距（纵向）：10.0
6. 舵机体开口：41 x 20（矩形开口，四角 R2）
7. 舵盘轴中心位于底板长边中心线上
8. 螺丝：M3

说明：MG996R 不同批次耳距略有差异，因此用长孔能更稳妥。

### A2. MG90S 安装模板（推荐用长孔）
1. 底板尺寸：42 x 22 x 4
2. 长孔数量：4
3. 长孔尺寸：8 x 2.6
4. 长孔中心距（横向）：27.5
5. 长孔中心距（纵向）：6.0
6. 舵机体开口：23 x 12（矩形开口，四角 R2）
7. 舵盘轴中心位于底板长边中心线上
8. 螺丝：M2 或 M2.5（可用 M3 但需放大孔位）

说明：MG90S 耳距也会有偏差，用长孔最稳。

## B. 打印分块草图说明（文字版）

### B1. 底盘
1. 分为上盖 + 下盖
2. 下盖承力，建议壁厚 3 壁以上
3. 上盖用于隐藏线束与电源

### B2. 立柱
1. 分为上下两段
2. 中间留 40 x 40 线槽
3. 连接面加 2 个定位榫

### B3. 胸腔
1. 左右壳体拼合
2. 后侧预留线束孔与固定点

### B4. 头部
1. 上盖 + 下盖
2. 眼睛孔位统一

### B5. 手臂
1. 上臂壳体 + 肘部座 + 前臂壳体
2. 每段壳体一侧留穿线槽

### B6. 夹爪
1. 夹爪底座 + 左指 + 右指
2. 指爪与导轨留 0.5~0.8 间隙

## C. 装配顺序图解（更细化）

1. 底盘下盖装好并固定电源座与开关
2. 立柱下段固定到下盖
3. 立柱上段拼接并锁紧
4. 胸腔左右壳体装配
5. 颈部垫块与头部安装
6. 左右肩部舵机居中并固定
7. 上臂连接肩部舵盘
8. 肘部舵机居中并固定
9. 前臂连接肘部舵盘
10. 夹爪舵机安装并连接指爪
11. 线束主干与分支整理
12. 接线完成后低速测试动作
13. 盖上外壳，最终调试

## D. 线序表（锁定版）
1. 左肩旋转 -> GPIO2
2. 左肘 -> GPIO3
3. 左夹爪 -> GPIO4
4. 右肩旋转 -> GPIO5
5. 右肘 -> GPIO6
6. 右夹爪 -> GPIO7

## E. 打印建议（锁定版）
1. 受力件 PLA Pro，外壳普通 PLA
2. 层高 0.2，外壳 3 壁，受力件 4 壁
3. 外壳填充 12~15，受力件 20~30
4. 重要孔位预留 0.3~0.5 补偿

## H. 舵机安装位置（结构版已预留空腔）
1. 肩部 MG996R：在胸腔左右侧面预留空腔与安装槽。
2. 肘部 MG996R：在上臂内部预留空腔与安装槽。
3. 夹爪 MG90S：在前臂内部预留空腔与安装槽。
4. 所有舵机空腔都预留 0.6 mm 公差，默认适配常见批次。
5. 开装前建议用卡尺复测舵机尺寸，必要时微调参数。
