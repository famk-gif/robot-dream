# 机器人打印与装配指南（锁定版）

## 参数清单（单位：mm）
1. 总高度：410（含轮子）
2. 底盘：260 x 260 x 40
3. 立柱：90 x 90 x 70
4. 胸腔：160 x 120 x 140
5. 颈部垫块：60 x 60 x 20
6. 头部：90 x 90 x 80
7. 肩轴中心距：180
8. 上臂长度：85
9. 前臂长度：85
10. 夹爪长度：30
11. 外壳厚度：2.4
12. 结构件厚度：4
13. 舵机安装板厚度：4
14. 螺丝规格：M3

## 图纸说明文本
1. 底盘分为上盖与下盖两件，均为 260 x 260 x 40，内部中空，底盘左右各一只轮子，轮径 60，并用侧向支架连接底盘。
2. 立柱分为上下两段，每段 90 x 90 x 35，内部留 40 x 40 线槽。
3. 胸腔分为左右壳体，内部留 120 x 80 的线束空间。
4. 颈部垫块用于把头部抬高，尺寸 60 x 60 x 20。
5. 头部上盖与下盖卡扣式连接，正面预留双眼开孔。
6. 肩部舵机座采用 MG996R 标准尺寸，舵盘中心与肩轴中心对齐。
7. 肘部舵机座采用 MG996R 标准尺寸，保证上臂与前臂同轴。
8. 夹爪舵机座采用 MG90S 标准尺寸，夹爪开合角度 35 到 45 度。
9. 所有连接孔统一 M3，孔位对称，孔距采用 20 或 30 的模数。
10. 外壳厚度 2.4，受力件厚度 4，避免薄壁变形。
11. 线束统一走胸腔后侧，正面保持干净。

## 线束整理方案
1. 电源线使用主干加分支方式，主干从底盘中心上行。
2. 肩部左右分叉后进入上臂内侧，肘部预留回环。
3. 线束与旋转部位保持 3 到 5 mm 余量，避免拉扯。
4. 所有线束固定点使用扎带或压线片。
5. 线束穿过孔位时加护线圈或胶管。

## 关键孔位与定位说明
1. 舵机孔位以实际舵机外形图或实物测量为准，避免不同批次误差。
2. 舵盘中心必须与关节旋转轴重合，避免偏心抖动。
3. 肩部、肘部孔位应对称，保证左右手臂一致。
4. 夹爪导向槽建议预留 0.5 到 0.8 的装配间隙。

## 打印建议
1. 受力件使用 PLA Pro，外壳使用普通 PLA。
2. 层高建议 0.2，外壳 3 壁，受力件 4 壁。
3. 填充率外壳 12 到 15，受力件 20 到 30。
4. 重要螺丝孔位建议预留 0.3 到 0.5 的补偿。
